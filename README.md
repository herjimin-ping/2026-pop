# 2026-pop# 📊 우리 동네 인구구조 시각화 (2026-pop)

행정안전부 주민등록 인구통계(1인세대) 데이터를 기반으로, 원하는 지역을 선택하면 연령대별 1인세대수 구조를 꺾은선 그래프로 한눈에 볼 수 있는 Streamlit 웹앱입니다.

바이브 코딩(vibe coding)을 배우면서 만든 첫 데이터 시각화 프로젝트예요. 🐣

---

## 🔍 사용한 데이터

- **출처**: 행정안전부 주민등록 인구통계
- **파일명**: `202606_202606_행정동별_성연령별_주민등록_1인세대수_월간.csv`
- **내용**: 2026년 6월 기준, 전국 행정동(읍면동) 단위 성별·연령대별 1인세대수
- **규모**: 3,912행 × 40열 (전국/시도/시군구/행정동 단위가 함께 포함되어 있음)
- **인코딩**: CP949(EUC-KR), 숫자에 천단위 콤마 포함 → 전처리 필요

## 🛠 기술 스택

- **Streamlit** — 웹앱 프레임워크
- **Pandas** — 데이터 전처리 (콤마 제거, 지역명/행정코드 분리, 단위 분류)
- **Plotly** — 인터랙티브 꺾은선 그래프

## ✨ 주요 기능 (app.py)

1. **데이터 자동 로드**: 저장소 내 CSV를 자동 탐색하거나, 사이드바에서 직접 업로드 가능
2. **지역 선택 (선택 + 검색 입력 동시 지원)**
   - 행정구역 단위 선택: 행정동 / 시군구 / 시도 / 전국
   - 시/도로 1차 필터링 → 목록 축소
   - `st.selectbox`의 타이핑 검색 기능으로 원하는 지역을 빠르게 찾기
3. **연령대별 인구구조 꺾은선 그래프**
   - 계/남/여 3개 라인을 동시에 표시 (0~9세 ~ 100세 이상, 11개 구간)
   - `hovermode="x unified"`로 마우스 오버 시 전체 연령대 값을 한 번에 확인
4. **요약 지표**: 총 1인세대수, 남/여 세대수를 상단 카드로 표시
5. **원본 수치 표**: 접었다 펼 수 있는 표로 정확한 숫자 확인 가능

## 🚀 로컬 실행 방법

    pip install -r requirements.txt
    streamlit run main.py

## ☁️ Streamlit Cloud 배포 방법

1. GitHub 저장소에 `main.py`, `requirements.txt`, CSV 데이터 파일을 함께 업로드
2. share.streamlit.io 접속 → New app
3. 저장소 / 브랜치 / `main.py` 지정 → Deploy

## 📚 참고: 외국인 인구 데이터 (추가 예정)

주민등록 인구는 원칙적으로 내국인만 집계되어, 외국인 통계는 별도 공공데이터가 필요합니다.

- 지역별 외국인 거주자 수 (시군구 단위): 공공데이터포털 - 법무부 등록외국인 시군구별 거주 현황
- 국적별 체류외국인 현황 (전국 단위): 공공데이터포털 - 법무부 체류외국인 국적 및 체류자격별 현황
